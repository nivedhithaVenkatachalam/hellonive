About Me

"Everything I know is but a drop in the ocean of knowledge."  - Albert Einstein

I am Nivedhitha Venkatachalam, graduating MS in Software Engineering at San Jose State University in December 2016.

I am doing my internship as a Software Engineer in a startup - Flow Labs, where I am building a data visualizing dashboard using ReactJS and MeteorJS. I did my summer internship as an user interface developer at Pulse Secure (San Jose, CA). I used HTML5, SCSS and AngularJS to create a new module for an enterprise application. I used Git for version control and JIRA for tracking the several features and bugs that I worked on. I also used Bower for front end package management and grunt to ensure the code is of production quality. I worked at Cognizant Technology Solutions (Chennai, India) for 2 years 9 months in application development and support. My interests are in web application development, data analytics and cloud technologies.

I am eager to take up new challenges which motivate me to learn new technologies and broaden my knowledge.
